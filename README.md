Proyecto: Operaciones con Arreglos en Java

🎯 Objetivo del proyecto
Este proyecto tiene como objetivo practicar el manejo de arreglos unidimensionales en Java.
El programa solicita al usuario ingresar valores para dos arreglos (array1 y array2). Luego, genera un tercer arreglo (array3) que contiene la diferencia entre los elementos de los dos primeros arreglos, mostrando el resultado y calculando el promedio de los valores obtenidos.

🧠 Descripción del código
Se declaran tres arreglos de tipo int con 7 posiciones cada uno.

Se utilizan bucles for para llenar array1 y array2 con números ingresados por el usuario.

El tercer arreglo (array3) se llena restando elemento a elemento los arreglos anteriores.

Se calcula la suma y el promedio de los resultados del tercer arreglo.

Finalmente, el programa muestra los valores de array3 y su promedio con dos decimales.

▶️ Instrucciones de ejecución
Clonar o descargar este proyecto en tu computadora.

Asegúrate de tener Java instalado (Java 8 o superior).

Compila el archivo desde la terminal o desde tu IDE favorito.

Ejecuta el programa:

bash
javac arraysAct1.java
java arraysAct1
Ingresa los valores solicitados para llenar los arreglos según las instrucciones en pantalla.

💡 Nota: si al ejecutar el programa obtienes un error, asegúrate de que el método main esté declarado correctamente como public static void main(String[] args).
